Automated installers

In the README.md in the repository github.com/goa-systems/goa.systems.powershell you can find automated installers for Eclipse, Java and Gradle. They will install the tools in subdirectories under

%LOCALAPPDATA%\Programs

Additionally enviornment variables are set. To use the installers simply copy the code block and paste it into PowerShell with [CRTL] + [V] (Right click paste does not work correctly).

Environment variables

The previous step installs the tools in the application directories and sets environment variables. They can be added to the PATH variable in the next step.

The PATH variable

The PATH variable contains a list of directories which are considered if a command is executed somewhere on the system.

Workspaces

  • Allow handling of different project types. For example .NET, Java, PHP, etc.
  • Keeps software development projects in one place. For example:
  • %USERPROFILE%\workspaces\java
  • %USERPROFILE%\workspaces\dotnet
  • %USERPROFILE%\workspaces\vscode
